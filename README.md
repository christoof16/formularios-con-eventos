# formularios-con-eventos

en este codigo el formulario tiene 3 input los cuales tienen un nombre,correo y materia
en la cual con un evento tipo click el formulario simula enviar los datos 
y ademas se genera algunas validaciones
(ejemplo) si el nombre tiene menos de 4 caracteres o mas de 40 carcteres
si el correo electronico tiene el @ y . si no no se envia los datos 
y si la materia tiene menos de 4 caracteres o mas de 40 caracteres si no no se envia
